Ce projet est une application web dont le but est l'exploration des villes du monde
des pokémons.
C'est un site web développé par html css js et php. Et une base de données PostgreSQL.
Il nécessite l'utilisation de wampserver et géoserver.
Toutes les informations nécessaires au fonctionnement de notre interface vous les
trouvez dans le fichier "guide_utilisation.pdf".
